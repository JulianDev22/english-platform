# Recursos Adicionales y Fuera de Alcance

## Incluido en el MVP
- Flashcards con SRS (SM-2)
- Quizzes de opción múltiple
- Rellenar espacios en blanco
- Matching (arrastrar/click)
- Ejercicios de escritura con rúbrica
- Roleplay con diálogos guiados
- Dictado con texto
- Grabación de audio para pronunciación (auto-evaluación)
- Progreso persistente SQLite
- PWA instalable

## Para versiones futuras
- **Text-to-Speech**: integración con Web Speech API o ElevenLabs para lecturas.
- **AI Corrector**: para evaluar respuestas abiertas en writing (OpenAI API).
- **Análisis de pronunciación**: comparación espectrograma.
- **Leaderboard personal**: estadísticas detalladas (racha, XP, tiempo de estudio).
- **Modo multidispositivo**: sincronización entre PC y móvil vía backend.
- **Importación/exportación de datos**: backup de progreso.
- **Compartir progreso**: enlace público de perfil de aprendizaje.
